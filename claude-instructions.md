# Claude를 사용한 개발 프로세스 가이드

## 🚨 중요: Claude에게 작업을 요청할 때

### 새로운 기능 개발 요청 시
```
다음 기능을 개발해주세요: [기능명]

반드시 아래 프로세스를 따라주세요:
1. 먼저 PRD 작성을 위해 관련 정보를 수집하고 정리해주세요
2. PRD 초안을 보여주고 검토 요청하세요
3. PRD 승인 후 기술 설계 계획을 작성해주세요
4. 설계 계획 검토 요청하세요
5. 설계 승인 후 코드 구현을 시작하세요

각 단계를 건너뛰지 말고 반드시 순서대로 진행해주세요.
```

### 버그 수정 요청 시
```
다음 버그를 수정해주세요: [버그 설명]

아래 절차를 따라주세요:
1. 버그 재현을 위해 관련 코드를 분석해주세요
2. 원인 분석 결과를 공유해주세요
3. 수정 계획을 설명해주세요
4. 계획 승인 후 코드 수정을 진행해주세요
```

## ✅ 항상 확인할 사항

1. **PRD 존재 여부**: 해당 기능에 대한 PRD가 있는지 확인
2. **설계 문서 존재 여부**: 승인된 설계 계획이 있는지 확인
3. **워크플로우 준수**: WORKFLOW.md에 명시된 프로세스 따르기
4. **문서 업데이트**: 구현 후 관련 문서 업데이트
